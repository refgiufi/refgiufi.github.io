# 🚀 Personal Portfolio Website

A modern, responsive portfolio website showcasing professional work, skills, and experience.

## ✨ Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Modern UI/UX** - Clean, minimalist design with smooth animations
- **Portfolio Showcase** - Interactive project gallery with filtering
- **Professional Sections** - Hero, About, Skills, Experience, Contact
- **GitHub Pages Ready** - Easy deployment and hosting

## 🛠️ Technologies

- HTML5 & CSS3
- Vanilla JavaScript
- Responsive Grid & Flexbox
- Modern CSS animations

## 🚀 Quick Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/refgiufi/refgiufi.github.io.git
   cd refgiufi.github.io
   ```

2. **Add your profile image**
   - Save your photo as `assets/images/profile.png`
   - Recommended size: 700x900 pixels

3. **Customize content**
   - Edit `index.html` for main content
   - Edit `portfolio.html` for projects
   - Update personal information and links

4. **Deploy to GitHub Pages**
   - Push to `main` branch
   - Enable GitHub Pages in repository settings
   - Your site will be live at `https://yourusername.github.io`

## 📱 Responsive Breakpoints

- **Desktop:** 1400px+
- **Tablet:** 768px - 1399px  
- **Mobile:** < 768px

## 🎨 Customization

The website is built with clean, modular CSS that's easy to customize:
- Colors and themes in CSS variables
- Typography with clamp() for responsive scaling
- Grid layouts for easy content organization
